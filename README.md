# AI Game Builder

AI Game Builder adalah web app yang membantu siapa saja membuat game dengan cepat dan mudah. Aplikasi ini menyediakan template mekanik dasar seperti race, shooter, dan parkour yang dapat diubah secara otomatis ke bahasa pemrograman atau engine apa pun melalui AI.

Pengguna cukup memilih template game, menulis prompt, dan AI akan menghasilkan kode game lengkap sesuai permintaan â€” misalnya Three.js, Python Pygame, HTML Canvas, dan banyak lainnya.

##  Fitur Utama

###  Template Game Siap Pakai
- Race 2D
- Race 3D
- Shooter 2D
- Parkour 2D
- Parkour 3D  
Template berisi **mekanisme inti** tanpa aset, sehingga mudah dikembangkan oleh pengguna.

###  AI Code Generator
- Mengambil template mekanik dasar
- Mengonversinya ke bahasa atau engine apa pun
- Menerjemahkan logika gameplay secara otomatis
- Menghasilkan file kode lengkap siap di-copy dan digunakan

###  Cepat & Sederhana
- Dibangun dengan HTML, CSS, dan JavaScript murni
- UI ringan & intuitif
- Tidak perlu instalasi tambahan

###  Backend AI Otomatis
Aplikasi terhubung ke backend AI yang menangani:
- Pemilihan template
- Transformasi kode
- Pemrosesan prompt user
- Output final yang konsisten


##  Teknologi yang Digunakan
- **Frontend:** HTML, CSS, JavaScript
- **AI Backend:** DeepSeek / Gemini API
- **Template Engine:** TXT-based modular template system

##  Cara Menggunakan
1. Buka aplikasi melalui browser
2. Pilih genre template game
3. Masukkan prompt (contoh: *"Buatkan versi Python Pygame"*)
4. Klik **Generate Code**
5. Kode game langsung muncul dan bisa di-copy atau diunduh

##  Cara Deploy
Aplikasi ini dapat di-host di:
- GitHub Pages
- Netlify
- Vercel
- Shared hosting biasa

Tidak membutuhkan build tools.
